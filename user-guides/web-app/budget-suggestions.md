# Budget Suggestions

**Find affordable dye alternatives**

The Budget Suggestions tool helps you find cheaper dyes that look similar to expensive options, using real-time market data from the Universalis API.

---

## How to Use

### 1. Select a Target Dye

Choose the dye you want alternatives for:
- **Search by name** - Type to find
- **From other tools** - Use "Find Budget Options" from any dye
- **Browse categories** - Select from list

### 2. Set Your Budget

Specify your price range:
- **Max price** - Highest you're willing to pay
- **Server** - Your FFXIV server/data center
- **Include crafted** - Whether to show craftable options

### 3. View Alternatives

Results show:
- **Similar dyes** - Ranked by visual similarity
- **Current prices** - Real-time market data
- **Savings** - How much cheaper than the target
- **Color difference** - How close the match is

---

## Understanding Results

### Price Data

| Column | Meaning |
|--------|---------|
| **Min Price** | Lowest current listing |
| **Avg Price** | Average of recent sales |
| **Listings** | How many are for sale |
| **Last Sale** | Most recent transaction price |

### Color Similarity

| Delta E | Meaning |
|---------|---------|
| 0-5 | Nearly identical |
| 5-15 | Very similar |
| 15-30 | Noticeably different |
| 30+ | Quite different |

## Tips

### Finding Good Deals

1. **Check multiple alternatives** - The cheapest may not be closest
2. **Compare across servers** - Prices vary by server
3. **Consider crafted dyes** - Often cheaper if you craft
4. **Wait for sales** - Prices fluctuate

### Color Compromises

1. **Slight shade differences** may not matter on some gear
2. **Metallic vs. matte** - Sometimes the cheaper version looks better
3. **Test in-game** - Some differences matter more than others

---

## Data Freshness

Market data is cached for performance:
- **Price data** - Updated every 5 minutes
- **Listings** - Real-time when clicked

If prices seem off, wait a few minutes and refresh.

---

## Server Selection

Choose your server/data center:
- **Specific server** - Most accurate for local prices
- **Data center** - Shows cross-world options
- **All** - Compare prices globally
