# Favorites & Collections

**Save and organize your dyes**

Favorites and Collections help you keep track of dyes you love and organize them for different purposes.

---

## Favorites

### What Are Favorites?

A quick-access list of your most-used or most-loved dyes. Limited to 20 favorites for easy browsing.

### Adding Favorites

From any tool, click the **★ Add to Favorites** button on any dye.

### Managing Favorites

Access your favorites from the sidebar:
- **View all** - See your complete list
- **Remove** - Click ★ again to unfavorite
- **Use** - Click to open in any tool

### Limits

- Maximum 20 favorites
- Remove old ones to add new
- Synced across devices when logged in

---

## Collections

### What Are Collections?

Custom groups of dyes organized by theme. Great for:
- Outfit projects
- Housing color schemes
- Job glamours
- Seasonal looks

### Creating Collections

1. Click **New Collection** in the sidebar
2. Give it a name
3. Optionally add a description
4. Start adding dyes

### Adding Dyes to Collections

From any tool:
1. Click **Add to Collection** on a dye
2. Select which collection (or create new)
3. Dye is added

### Managing Collections

- **Rename** - Click the edit icon
- **Delete** - Remove entire collection
- **Reorder** - Drag and drop dyes
- **Remove dyes** - Click X on individual dyes

### Limits

- Maximum 50 collections
- Maximum 20 dyes per collection
- Synced when logged in

---

## Syncing

### Logged In

When authenticated with Discord:
- Favorites sync across devices
- Collections sync across devices
- Available on web and Discord bot

### Not Logged In

Data is stored locally:
- Stays in browser only
- Cleared if you clear browser data
- Not synced to Discord bot

## Tips

### Organization Ideas

| Collection Name | Use Case |
|-----------------|----------|
| "Tank Glamours" | Dyes for tank jobs |
| "Beach House" | Housing project |
| "Seasonal Events" | Holiday colors |
| "Top Picks" | Your very favorites |
| "To Try" | Dyes to test in-game |

### Workflow Tips

1. **Star immediately** - Favorite dyes as you find them
2. **Sort later** - Move favorites into collections when you have time
3. **Project collections** - Create one for each active glamour project
4. **Archive** - Keep finished project collections for reference

---

## Discord Bot Sync

Your favorites and collections are also available via the Discord bot:

| Command | Description |
|---------|-------------|
| `/favorites` | View and manage favorites |
| `/collection list` | List your collections |
| `/collection show` | View collection contents |

Changes sync automatically between web and Discord.
