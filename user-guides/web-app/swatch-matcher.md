# Swatch Matcher

**Match character colors to FFXIV dyes**

The Swatch Matcher helps you find FFXIV dyes that match your character's customization colors - hair, eyes, skin, tattoos, face paint, and more.

> **Note**: This tool was previously called "Character Color Matcher" in v3.x.

---

## How to Use

### 1. Select Your Character

Choose your character's race and clan:

**Available Races & Clans:**

| Race | Clans |
|------|-------|
| **Hyur** | Midlander, Highlander |
| **Elezen** | Wildwood, Duskwight |
| **Lalafell** | Plainsfolk, Dunesfolk |
| **Miqo'te** | Seeker of the Sun, Keeper of the Moon |
| **Roegadyn** | Sea Wolf, Hellsguard |
| **Au Ra** | Raen, Xaela |
| **Hrothgar** | Helion, The Lost |
| **Viera** | Rava, Veena |

### 2. Choose a Color Category

Select which type of color to match:

| Category | Description |
|----------|-------------|
| **Eye Colors** | Iris colors available for your race |
| **Hair Colors** | Hair color options |
| **Skin Colors** | Skin tone options |
| **Highlight Colors** | Hair highlight options |
| **Lip Colors** | Lip color options (light/dark themes) |
| **Tattoo Colors** | Limbal rings and racial tattoos |
| **Face Paint** | Face paint color options |

### 3. Select a Color

Browse the color grid showing all available colors for your selection. Click any color to see matching dyes.

### 4. View Matches

For each selected color:
- **Best match** - The closest FFXIV dye
- **Top alternatives** - Other close options
- **Delta E scores** - Match accuracy

## Color Grid

The color grid displays all available options based on your selections:

- Colors are pulled from FFXIV's character creation data
- Grid layout matches the in-game character creator
- Hover over colors to see hex values
- Click to find matching dyes

## Why Match Character Colors?

### Coordinate with Your Character
Find dyes that match your character's:
- **Hair color** for seamless wig/hat coloring
- **Eye color** for accessories and accents
- **Skin tone** for natural-looking gear

### Create Themed Glamours
Build outfits that complement your character's features:
1. Match a dye to your hair color
2. Use it for chest piece accents
3. Create a cohesive look

### Planning Before Character Creation
If you're making a new character:
1. Browse available colors per race
2. See what dyes would match each option
3. Plan your glamour before committing

## Tips

- **Some colors have no exact match** - FFXIV dyes don't cover every possible shade
- **Check multiple alternatives** - The 2nd or 3rd match might work better in-game
- **Lighting matters** - Colors appear differently in various zones
- **Test in-game** - Use the dye preview before committing

---

## Understanding Delta E

Delta E measures how close a dye is to your selected color:

| Delta E | Meaning |
|---------|---------|
| 0-1 | Virtually identical |
| 1-2 | Very close, minor difference |
| 2-10 | Noticeable but similar |
| 10+ | Visibly different |

Lower scores mean better matches!
